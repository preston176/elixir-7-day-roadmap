# Contributing

Thanks for wanting to contribute! This repository is meant to be a friendly learning resource.

Guidelines:

- Keep language simple and examples runnable.
- Target content for JavaScript/Next.js developers learning Elixir.
- Add small, focused improvements as separate PRs (one topic per PR).
- For code changes, include short tests or a verification step in the README.

Report issues via GitHub Issues or open a PR with suggested changes.
